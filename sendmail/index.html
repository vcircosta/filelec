Présentation de sendmail

sendmail est un programme simple permettant d'envoyer des messages électroniques en local via smtp et PHP. La plupart des hébergeurs possèdent leur propre sendmail à eux.

Pourquoi utiliser sendmail ?

sendmail, lui, possède une utilisation assez simple et fonctionnelle.

Téléchargement de sendmail

  • download sendmail.zip-> https://www.glob.com.au/sendmail/
  • Enregistrer l'archive sur le Bureau
  • Décompresser l'archive en créant un nouveau dossier nommésendmail
  • Déplacer ce dossier à la racine du serveur local. PAS DANS LE DOSSIER DES PROJETS !!

Configuration de sendmail

  • Dans le dossiersendmail, ouvrir le fichiersendmail.iniavec un éditeur de code
  • Saisir les informations suivantes :
    • smtp_server = smpt.gmail.com
    • smtp_port = 587
    • ;default_domain = gmail.com
    • auth_username = (saisir l'adresse du compte non sécurisé)
    • auth_password = (saisir le mot de passe du compte non sécurisé)
    • force_sender = (saisir l'adresse du compte non sécurisé)
  • Renseigner le chemin d'accès desendmail.exedans le fichierphp.inidu server local :sendmail_path.
    LAMP : etc -> php -> Version -> apache2 -> php.ini (pas sûr) 
    sendmail_path = "/var/sendmail/sendmail.exe" (pas sûr)
    MAMP : Applications -> MAMP -> conf -> phpVersion -> php.ini 
    sendmail_path = "/MAMP/sendmail/sendmail.exe"
    WAMP : C -> wamp64 -> bin -> apache -> apacheVersion -> bin -> php.ini 
    sendmail_path = "C:\wamp64\sendmail\sendmail.exe"
    XAMPP : C -> xampp -> php -> php.ini 
    sendmail_path = "C:\xampp\sendmail\sendmail.exe"
  • Désactiver le port 25 du smpt dons le fichierphp.inidu serveur local : il suffit juste d'ajouter un ; avant smtp 

    ;smtp_port = 25
  • REDEMARRER LE SERVEUR LOCAL !!

Mot de passe oublié

1 - Création de la tablerecuperation

Cette table va servire à envoyer un code de vérification à l'adresse email fournis, pour ensuite réinitialiser le mot de passe de l'utilisateur. 

Drop table if exists recuperation;
Create table if not exists recuperation (
	idrecup int(11) not null auto_increment,
	email varchar(100) not null,
	code int(11) not null,
	confirme int not null default 0,
	primary key (idrecup)
) ENGINE=InnoDB;

2 - Création de la fonctionselectWhereRowCount()

Cette fonction va permettre de vérifier si l'adresse email saisi existe bien dans la tableuser. Vous pouvez l'appelé comme vous vous voulez, mais n'oubliez pas de modifier son nom dans le fichiergestion_mdp.php.

modele/modele.class.php 
public function selectWhereRowCount($where) {
	if ($this->pdo != null) {
		$champs = array();
		$donnees = array();
		foreach ($where as $key => $value) {
			$champs[] = $key. " = :".$key;
			$donnees[":".$key] = $value;
		}
		$chaineWhere = implode(" AND ", $champs);
		$requete = "SELECT * FROM ".$this->uneTable." WHERE ".$chaineWhere;
		$select = $this->pdo->prepare($requete);
		$select->execute($donnees);
		$select->fetch();
		return $select->rowCount();
	} else {
		return null;
	}
}
controleur/controleur.class.php 
public function selectWhereRowCount($where) {
	return $this->unModele->selectWhereRowCount($where);
}
